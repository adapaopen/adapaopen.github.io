# 命名 意义 方法
1 __dirname __filename 打印目錄 console.log(__dirname); //不包含文件名的路徑 console.log(__filename);//包含文件名的路徑
2 global 全局變量
3 module.exports 模塊 var cou=func(){};module.exports=cou;//包COU做成模塊 //requirevar var counter = require('./app'); //在new的文件導入模塊文件的地址 console.log(counter(['Help','皮醬','Like+']));//調用模塊 OR module.exports = { counter:counter, adder:adder, pi:pi }
4 setImmediate() 異步 ES6: setImmediate(() =>{ console.log('註冊事件'); console.log(msg); }); ES5: setImmediate(function(){coding;};)
5 Events 事件模塊 時間流程：引入模塊->創建EventEmitter對象->註冊事件->觸發事件代碼塊
6 FS 文件系統之讀寫文件 fs.readFile讀取文件 fs.writeFile寫入文件 流程:引入fs模塊->調用方法->異常捕獲 Coding
7 FS 文件系統之文件夾文件操作 創建文件夾fs.mkdir 刪除文件夾fs.rmdir 刪除文件fs.unlink 流程:引入fs模塊->調用方法->異常捕獲 Coding
8 HTTP HTTP模塊 http.createServer(function(req;res){};);//創建服務器[http.createServer([requestListener]) 了解更多] Coding
9 createReadStream;createWriteStream;pipe 讀寫數據流 查看案例
